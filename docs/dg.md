## Issues

The following guidelines on **labelling** shall be followed when GitHub Issues are raised:

features
{: .label .label-green}
* new sections

bugs
{: .label .label-red }
* bugs (e.g. dead links)

documentations
{: .label .label-blue}
* new pages in a section

ui
{: .label .label-yellow}
* UI optimisations

## Thumbnails

The width for a thumbnail image should be 720px.
